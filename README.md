Project Title

Data Cleaning, Transformation, and Formatting in Microsoft Excel

Objective

The purpose of this project is to perform data cleaning, transformation, and formatting operations on a product dataset using Microsoft Excel. The tasks focus on improving data quality, consistency, readability, and visualization.

Tasks Performed
1. Handling Missing Values
Price Column
Checked the Price column for missing values.
Identified products with missing price information.
Suggested handling methods:
Replace missing values with the average price of similar products.
Obtain the correct value from the original data source.
Mark records for further review if the correct value is unavailable.
Category Column
Checked the Category column for missing values.
Suggested handling methods:
Assign the most appropriate category based on the product name.
Use the most frequently occurring category when applicable.
Label as "Unknown" if the category cannot be determined.
2. Correcting Inconsistent Data
Product Name Column
Reviewed the Product Name column for inconsistent text formats.
Standardized capitalization and naming conventions.
Category Column
Identified spelling mistakes and typographical errors.
Used the Find and Replace feature to correct misspellings and maintain consistency.

3. Removing Duplicates
Checked the dataset for duplicate records.
Used Excel's Remove Duplicates feature.
Removed duplicate rows based on the complete row data.
4. Splitting and Merging Data
Splitting Product ID
Split the Product ID column into:
Manufacturing Date
Country Code
Removed unnecessary characters where required.
Merging Columns
Combined Brand Name and Product Name into a new column called Product Brand.
Used concatenation techniques to merge the values.
5. Number Formatting
Price Column
Formatted the Price column using Currency format.
Applied the appropriate currency symbol and decimal places.
Manufacturing Date Column
Formatted dates using the custom format:
DD-MM-YYYY

Example:

28-01-2024
15-03-2024
05-12-2024
6. Conditional Formatting
Price Column
Applied Data Bars or Color Scales to visually represent price variations.
Enabled quick comparison of product prices.
Category Column
Created a custom conditional formatting rule to highlight cells containing:
Electronics
Applied a distinct fill color to improve visibility.
